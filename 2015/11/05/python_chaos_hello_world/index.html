Step 4

我们知道 __import__('os') 这里的 os 是个字符串，直接写个 os 在那里容易就被人看出是做什么的了，那么我们就可以使用字符串拼接的方法来构造 os 和 write 这两个字符串。

func_code.co_lnotab 是查找函数中代码对象的行号表，我们用的是匿名函数，就没有行号，于是返回的就是空
